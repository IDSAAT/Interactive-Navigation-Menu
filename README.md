Interactive-Navigation-Menu.
A responsive, fixed navigation menu that changes appearance on scroll and hover. Built with vanilla HTML, CSS, and JavaScript.

📌 Live Demo

[View Live Demo](https://your-demo-link.netlify.app)

🎯 Features

1.Fixed Position: Menu stays visible at the top while scrolling
2. Scroll Effect: Background changes from transparent to solid color after scrolling 50px
3. Hover Effects: Menu items lift and change color on hover
4. Responsive Design: Works on desktop, tablet, and mobile devices
5. Cross-Page Consistency: Same navigation appears on all pages (Home, About, Services, Contact)

🛠️ Technologies Used

| Technology | Purpose 
|------------|---------
| HTML5      | Page structure.
| CSS3       | Styling, transitions, responsive layout. 
| JavaScript | Scroll detection, dynamic class toggling.
